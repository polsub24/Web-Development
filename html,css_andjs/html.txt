<h1> to <h6>: These tags are used to create different levels of headings - h1 is the biggest and h6 is the smallest one
<a>: This tag is used for hyperlinks.
<img>: This is used to embed images in the webpage.
<!DOCTYPE html>: Identifies the document as an HTML5 document.
<html>: Encloses the HTML document.
<head>: Contains meta-information about the HTML document
<title>: The content within the title tag is used as the page title. Present inside the head section.
<body>: Hosts the content visible to web users — text, images, links, etc.

<!-- text --> is the comment syntax in HTML.
<a href="https://www.example.com">Visit Example.com</a> : To specify the link's destination.

<ul> tag signifies the list, and the <li> tag encloses each item.
<ul> for bulleted list
<ol> for numbered list
Nesting list also possible
In HTML, tables use the <table> tag. The <tr> tag facilities new rows, and <td> is for cells within the rows. Headers use the <th> tag.

Commands:-
<b> makes text bold.
<i> italicizes text.
<u> underlines text.
<s> strikes through text.
<br> inserts a line break.
<hr> draws a horizontal line.   
&lt; for <
&gt; for >
&amp; for &
&deg; for °
&copy; for ©
